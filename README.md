## Project Overview

This is a **full-stack stock trades dashboard** application built with:

- **Frontend:** ReactJS  
- **Backend:** FastAPI  
- **Database:** Supabase  
- **Hosting:** Render  
- **Containerization:** Docker  

The application allows users to view stock trades in a dashboard interface, leveraging React for interactive UI, FastAPI for backend API services, and Supabase as the database. The project demonstrates a full-stack workflow, including containerization for consistent deployment environments.

---

## Features

- Interactive dashboard for stock trades  
- API integration with FastAPI backend  
- Supabase database for data storage and retrieval  
- Dockerized frontend and backend for easy deployment  
- Hosted live version on Render  

---

## Challenges & Learnings

While developing this project, I faced several challenges integrating **ReactJS + FastAPI + Supabase**, particularly in managing CORS, authentication, and database connectivity. Overcoming these challenges helped me deepen my understanding of these technologies and full-stack development patterns.
